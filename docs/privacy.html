INTEGRITETS­POLICY

Senast uppdaterad: juni 2026

SAMMANFATTNING

LockIn samlar inte in någon data om dig. All din information lagras lokalt på din enhet och delas aldrig med tredje part, annonsörer eller molntjänster.

OM APPEN

LockIn är en iOS-träningsapp utvecklad av Josef Axén. Appen hjälper dig att logga träningspass, följa progress och sätta personliga rekord.

Appen kräver inget konto och kopplas inte till några externa servrar. Din data är din.

DATA SOM LAGRAS LOKALT

Följande information sparas uteslutande på din enhet via en lokal SQLite-databas:

  • Träningspass — övningar, set, reps, vikt och tidsstämplar
  • Träningsmallar och program du skapar
  • Kroppsvikt och kroppssammansättning (om du väljer att logga det)
  • Träningsfoton (om du väljer att lägga till dem)
  • Appinställningar

Ingen av denna data lämnar din enhet.

APPLE HEALTH (HÄLSOAPPEN)

LockIn kan integreras med Apple Health för att läsa hälsodata som hjälper dig förstå din återhämtning bättre. Appen skriver ingen data tillbaka till Health.

Följande Health-kategorier kan läsas, om du beviljar tillstånd:

  • Hjärtrytmvariabilitet (HRV) — för återhämtningsbedömning
  • Vilopuls — för att spåra kardiovaskulär hälsa
  • Sömnanalys — för att bedöma återhämtning efter sömn
  • Stegräknare och aktiv energi

Du väljer själv vilka Health-kategorier LockIn får tillgång till. Tillståndet kan återkallas när som helst via Inställningar → Hälsa → Apps på din iPhone.

Health-data används enbart för att visa information i appen och delas aldrig med tredje part.

INGEN INSAMLING, INGA SPÅRNING

LockIn:

  • Använder inga analytiksverktyg (Mixpanel, Firebase, Amplitude eller liknande)
  • Skickar ingen telemetri eller krasch­rapporter till externa servrar
  • Visar inga annonser och delar ingen data med annonsörer
  • Kräver inget konto och samlar inte in e-post eller personuppgifter
  • Kommunicerar inte med några externa API:er

TA BORT DIN DATA

All data raderas automatiskt om du avinstallerar appen. Du kan även radera all lokal data inifrån appen via Inställningar → Återställ data.

Eftersom ingen data skickas till externa servrar finns det inget att begära borttagning av.

BARN

LockIn riktar sig inte till barn under 13 år. Vi samlar inte medvetet in information från barn.

FÖRÄNDRINGAR I POLICYN

Om policyn ändras uppdateras datum på denna sida. Större förändringar kommuniceras via app-uppdateringar.

KONTAKT

Frågor om integritet? Kontakta oss via supportsidan eller öppna ett ärende på GitHub.
